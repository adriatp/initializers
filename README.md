# Initializers

## Install ArchLinux
```bash
loadkeys es
curl -L https://raw.githubusercontent.com/adriatp/initializers/main/arc/install.sh > install.sh
chmod 777 install.sh
./install.sh
```

## Install yay

```bash
curl -L https://raw.githubusercontent.com/adriatp/initializers/main/arc/yay.sh > yay.sh
chmod 777 yay.sh
./yay.sh
```
